## 📺 유튜브 연계형 AI 자기주도 학습 플랫폼
**(YouTube-Linked AI Self-Directed Learning Platform)**

### 🎯 프로젝트 목표

본 프로젝트는 **대학 강의 계획서**와 **유튜브**를 연계하여 학습 자료 검색 시간을 단축하고, **AI(LLM)**를 활용한 요약 및 퀴즈 생성을 통해 학습 효율을 극대화하는 **자기주도 학습 웹 애플리케이션** 구축을 목표로 합니다.

1.  **학습 접근성 강화**: 대학 시간표 및 강의 계획서와 연동하여 클릭 한 번으로 관련 학습 영상을 추천합니다.
2.  **AI 학습 보조**: 영상의 자막을 추출하고 LLM을 통해 **핵심 요약** 및 **이해도 점검 퀴즈**를 자동 생성합니다.
3.  **LMS 기능 통합**: 학습 메모, 영상 저장, 할 일 관리(Checklist)를 통해 체계적인 학습 관리를 지원합니다.

---

### 💡 주요 기능 및 특징

#### 1. 유튜브 기반 AI 학습 엔진

유튜브 API와 로컬 LLM을 결합하여 영상 검색부터 학습 평가까지의 과정을 자동화했습니다.

| 모듈 | 설명 |
| :--- | :--- |
| **Smart Search** | 조회수, 좋아요, 댓글, 키워드 연관성을 종합한 자체 알고리즘으로 최적의 학습 영상을 추천합니다. |
| **Auto Summary** | `youtube_transcript_api`로 자막을 추출하고, 로컬 LLM(Llama)이 핵심 내용을 3줄로 요약합니다. |
| **AI Quiz Gen** | 요약된 내용을 바탕으로 **JSON Schema**를 적용하여 정형화된 객관식 퀴즈(5문항+해설)를 생성합니다. |

#### 2. 대학 생활 밀착형 연동 & LMS

| 구분 | 기능 | 내용 |
| :--- | :--- | :--- |
| **Academic Link** | 강의 계획서 연동 | 시간표 과목 클릭 시 주차별 학습 목표/내용 확인 및 관련 유튜브 영상 즉시 검색 연동. |
| **LMS Tools** | 학습 관리 | **영상 저장고**: 중요 영상 보관 및 관리.<br>**Study Note**: 시청 중 메모 작성 및 .txt/.docx 다운로드.<br>**Checklist**: 날짜별 공부 할 일(To-Do) 관리. |

### 🧑‍💻 담당 역할 (Key Contributions)

**1. 프로젝트 기획 및 아키텍처 설계**
* 대학 전공 공부와 유튜브의 연결성을 고려하여 **'강의 계획서 → 유튜브 검색 → AI 요약/퀴즈'** 파이프라인 기획.
* Streamlit 기반의 SPA(Single Page Application) UI 구조 및 효율적인 세션 관리(Session State) 설계.

**2. LLM 기능 구현 및 최적화**
* **로컬 LLM 통합**: `llama-cpp-python`을 활용하여 경량화된 로컬 모델(`colab.py`) 구동 환경 구축.
* **프롬프트 엔지니어링**:
    * **요약**: 긴 자막 텍스트에서 핵심 주제만 추출하도록 시스템 프롬프트 최적화.
    * **퀴즈 생성**: 시스템 안정성을 위해 **JSON Schema**를 강제하여, 문제/보기/정답/해설이 명확히 구분된 데이터 출력 구현.
